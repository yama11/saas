
# AppFormDialog

## # Overview

表单弹窗组件

## # Props

```text
url           string?=''       表单提交地址
value         object?=null     后端最后提交的数据
object        string           表单页标题主体
id            number?=null     数据编辑需要传入
visible       boolean?=false   表单是否可见
```
